## About The Project

An attempt to create a game engine



## Getting Started

### Requirements
* Vulkan 1.1



### Dependencies
* VulkanSDK with VMA (https://vulkan.lunarg.com/sdk/home)



### How to build

1. Install dependencies

2. Clone the repo
  ```sh
  git clone https://github.com/RexarX/HeliosEngine.git
  cd HeliosEngine
  git submodule update --init --recursive
  ```
3. Build with cmake
  ```sh
  mkdir build
  cd build
  cmake ..
  ```
