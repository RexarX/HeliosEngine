Detailed Description

template<typename Iter>
requires ReverseAdapterRequirements<Iter>
class helios::utils::ReverseAdapter< Iter >

Adapter that iterates through elements in reverse order.

Requires a bidirectional iterator. Uses operator-- to traverse backwards.

IterType of the underlying iterator
