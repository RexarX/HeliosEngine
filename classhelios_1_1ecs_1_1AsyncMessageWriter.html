Detailed Description

template<AsyncMessageTrait T>
class helios::ecs::AsyncMessageWriter< T >

Type-safe writer for async messages.

Async messages use a lock-free concurrent queue, making writes thread-safe. This is suitable for messages produced from multiple threads concurrently (e.g., I/O callbacks, background workers). AsyncMessageWriter is intended to be short-lived (function-scoped).

Note
Thread-safe.
