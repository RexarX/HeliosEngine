Detailed Description

template<ResourceTrait T>
class helios::ecs::TryInsertResourceCmd< T >

Command to try insert a resource (only if missing).

Inserts resource of type T into the world during execution only if it doesn't already exist. If resource already exists, it will be left unchanged.
