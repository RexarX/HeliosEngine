Detailed Description

Free list allocator with best-fit strategy.

General-purpose allocator that maintains a free list of availablememory blocks. Supports arbitrary allocation sizes and orders. Uses a best-fit allocation strategy to minimize fragmentation.

Each free block contains a header with size and pointer to the next free block. Allocated blocks also store a header with size information for deallocation.

Supports arbitrary allocation and deallocation order. May suffer from fragmentation over time with varied allocation patterns. Adjacent free blocks are coalesced to reduce fragmentation.

Note
Thread-safe.
