Detailed Description

Type-erased entry for sparse-set component storage.

Stores a SparseComponentStorage<T> inside a TypedBuffer along with type-erased function pointers for operations that need to be performed without knowing the concrete type (e.g., clearing all storages, removing an entity from all storages).
