Detailed Description

template<typename... Ts>
class helios::ecs::ComponentBundle< Ts >

Owning collection of components that can be added or removed as one ECS operation.

Bundles may contain components and other bundles. Nested elements are flattened depth-first in declaration order when consumed by the ECS.
